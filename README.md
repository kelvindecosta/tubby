# Tubby

A utility for the [Genshin Impact Housing system](https://genshin-impact.fandom.com/wiki/Housing).

## Installation

```bash
pip install tubby
```

> ### Verification
>
> To verify that Tubby is installed, run:
>
> ```bash
> python -c "import tubby"
> ```
